## 📚 Structure des Exercices

### **01 - Variables et Calculs**
Manipuler les variables, les types de données et effectuer des calculs mathématiques simples.
- aire_cercle.c
- conversion_vitesse.c
- prix_ttc.c
- hypotenuse.c
- salaire_net.c

### **02 - Conditions**
Utiliser les structures conditionnelles (if, else, switch) pour prendre des décisions.
- categories_age.c
- calculatrice.c
- voyelle_consonne.c
- triangle.c
- jours_mois.c

### **03 - Boucles**
Implémenter des boucles (for, while, do-while) pour répéter des actions.
- nombres_pairs.c
- somme_n.c
- triangle_etoiles.c
- compteur_pos_neg.c
- puissance.c

### **04 - Fonctions**
Créer et utiliser des fonctions pour modulariser le code.
- est_pair.c
- valeur_absolue.c
- conversion_temperature.c
- compter_chiffres.c
- afficher_etoiles.c

### **05 - Tableaux 1D**
Travailler avec les tableaux à une dimension.
- somme_moyenne.c
- notes_dessus_dessous.c
- tri_bulles.c
- nombres_pairs_tab.c
- copie_tableau.c

### **06 - Tableaux 2D**
Manipuler les tableaux à deux dimensions (matrices).

### **07 - Chaînes de Caractères**
Gérer les chaînes de caractères et les opérations sur celles-ci.
- longueur_prenom.c
- comparaison_mots.c
- carte_identite.c
- mot_inverse.c
- mot_de_passe.c

### **08 - Récursivité**
Comprendre et implémenter des fonctions récursives.
- factorielle.c
- somme_recursive.c
- fibonacci.c
- puissance_recursive.c

### **10 - Valeurs Globales**
Utiliser les variables globales et comprendre leur portée.

### **11 - Pointeurs**
Maîtriser les pointeurs, les références et la gestion mémoire.
